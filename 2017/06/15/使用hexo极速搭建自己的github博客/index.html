使用hexo极速搭建自己的github博客

发表于 2017-06-15 | | 阅读次数

背景

大学时期曾经使用学校实验室的Linux服务器结合开源的wordpress工具，创建自己的博客，网上找模板，美化自己的博客主页，找各种免费云存储服务器，CDN加速服务等优化个人博客的加载速度,当时很享受自己建站的成就感。因为是用的学校的服务器，临近毕业时，忙于各种琐事，忘记备份博客资料和迁移新的服务器，大学期间写的那些博客也全部都消失殆尽了。工作后一直想着重新搭建一个自己的博客，但是一直没有付诸行动。最近被一些hexo搭建的技术博客吸引到了，就去官网上看了下文档和介绍，发现相当简单和方便，结合github,也省去了租赁服务器的麻烦。
这里记录下安装和使用过程，给即将开始搭建博客的小伙伴们做个参考

首先注册自己的Github账号，这里就不讲了

创建仓库用于托管博客

登录Github账号后，点击的右上方选择New repository创建仓库。

注意仓库名称格式一定不能错：Github昵称.github.io,这个网址以后就是别人访问你博客的地址

安装git

安装git方法这里就不再详述，如果你的电脑还没有安装，可以参考网上的安装教程

安装nide.js

安装hexo
