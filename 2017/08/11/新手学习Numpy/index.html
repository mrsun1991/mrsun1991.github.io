新手学习Numpy

发表于 2017-08-11 | | 阅读次数

NumPy是Python科学计算的基础包。它提供了多维数组对象、基于数组的各种派生对象（例如，masked Array, 矩阵）。除此之外，还提供了各种各样的加快数组操作的例程，包括数学基本计算、逻辑、图形操作、排序、选择、输入输出，离散傅立叶变换、基础线性代数、基础统计操作、随机仿真等等。
本篇文章主要讲一些Numpy的基础用法。

1.属性

首先创建一个numpy对象：

输出：

[[1 2 3]
 [2 3 4]
 [4 5 6]]

这样我们就创建了一个二维的3*3矩阵

（1）NDim（维数）

输出：

NDim: 2

我们可以看到NDim属性可以获取当前numpy对象的维数为二维

（2）shape（形状）

输出：

shape: (3, 3)

shape属性可以获得当前矩阵的形状为：3行3列

（3）size（大小）

输出：

size: 9

size可以获取矩阵的大小，即 3*3 = 9

2.Numpy创建array

输出：
[[ 2 23 4] [ 2 32 4]]
我们可以看到这样一个Numpy的array就创建完成了，从打印来看，这个array和Python的区别，它是没有逗号隔开。
创建的时候我们可以通过使用dtype属性来设置它的数据类型.

Numpy数据类型：

说道数据类型的，我们来看下它的类型转换（astype）

int64

 [  2.  32.   4.]]

可以看到，通过astype()函数，就把a的数据类型由int转换为了float32

输出（2x3）：

定义有序矩阵

输出（从10到20，步进为3的矩阵）：

[10 13 16 19]

输出（0~11的3x4阶矩阵）：

[[ 0  1  2  3]

注意：这里的arange（）中参数为前闭后开区间，即包含起始值，不包含结尾值；并且这个区间的定义一定要完全等于你定义的矩阵所需要的值

等差矩阵

我们可以看到cumsum方法可以获得当前行、列、及整个矩阵的累加后的矩阵

矩阵的截取：clip(a, a_min, a_max, out=None)

小于a_min的值全部置为a_min,大于a_max的值全部置为a_max，其他值不变

输出：

[[2 3 4 5]
